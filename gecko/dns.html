This document aims to clarify the naming scheme used in the Gecko project.

Naming scheme

Since physical hosts cannot be easily changed, they get assigned proper hostnames.

Virtual hosts get assigned random hostnames.

Regional DNS Zones

Everything that provides infrastructure to gecko should be under gecko.oxide.one

Within that, the naming for the gecko project should be within regional zones.
