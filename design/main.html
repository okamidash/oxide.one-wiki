Machine Class

The following is a table describing the classes of machines that are used within oxide.one.

Class A1

Class A1 is a 1U Server, typically installed in a Supermicro CSE-815 Chassis. The purpose of Class A1 is to provide a fair amount of compute and network power inside a small frame. A1 Class servers will be used to host mission critical services, such as DNS or Kubernetes Master nodes.

A1 Servers are designed to have a high uptime, so the CPU model is kept to a relatively low spec to allow for overkill cooling, and low power draw.
